REAL-TIME TASK SCHEDULING ON STM32F401RE EMBEDDED SYSTEMS

STM32 WATER PUMP SIMULATOR

Embedded C STM32Cube IDE

Consisted of multiple parts including PWM motor, potentiometer, ultrasonic sensor, etc., this project was ran upon a NUCLEO-F401RE and a custom-soldered PCB. Realizing the features of a water pump simulation. Incorporated PWM-controlled motors for precise pump operation, an ultrasonic sensor for accurate water level detection, and a seven-segment display for real-time data visualization. Established UART in addition for communication with a PC for interactive data visualization and configured JTAG pins for effective debugging and programming.

VIRTUAL GRAPPLER PLATFORM

VHDL

This is a project that simulates a platform that enables moving the grappler to a certain coordinate and activate it. This project contains several blocks including 2 registers, 2 binary counters, and 2 Compx4's. It also supports converting hexadecimal number to a 7-segment number, by including a number of 2 hex to 7-segment decoders. It also includes a reset features, where it is allowed to set eveerything to its normal state.
